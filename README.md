# BBS Barber & Salon Supply

Marketing and showcase website for **BBS Barber & Salon Supply** — a brick-and-mortar barber and salon supply store in Los Angeles. This is not an e-commerce site; the goal is to drive foot traffic to the physical store.

## Business Info

- **Address:** 202 E Martin Luther King Jr Blvd, Ste B, Los Angeles, CA 90011
- **Phone:** (323) 715-4214
- **Hours:** Mon–Fri 8:30am–5:00pm, Sat 9:00am–1:00pm, Sun Closed

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Static JSON data files (no backend)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Data Files

- `data/business-info.json` — Real business NAP, hours, coordinates, social links, reviews
- `data/products.json` — Placeholder product catalog (swap with real catalog when ready)

## Deploy

Deploy to [Vercel](https://vercel.com) with zero config. Optionally set:

```
NEXT_PUBLIC_SITE_URL=https://your-domain.com
```

## Build

```bash
npm run build
npm start
```
